1. 좌측 사이드바 (핵심 탐색 영역) 🧭
이곳에서 사용자는 원하는 정보를 빠르고 쉽게 찾을 수 있어야 합니다.

학기 주차 네비게이션:

'1주차'부터 '15주차'까지의 목록을 세로로 나열합니다.

현재 진행 중인 주차는 특별한 색상이나 굵은 글씨로 강조 표시하여 사용자가 바로 알 수 있게 합니다. (예: 8주차 (중간고사))

각 주차 옆에 해당 주의 시작일과 종료일을 작은 글씨로 함께 표시해주면(예: 10/20 ~ 10/26) 더욱 편리합니다.

미니 월력:

작은 월간 달력을 넣어 전체적인 월의 흐름을 파악하고, 특정 날짜로 빠르게 이동할 수 있는 기능을 제공합니다.

일정 필터링:

체크박스를 이용해 특정 종류의 일정만 골라볼 수 있게 합니다.

[ ] 전체

[ ] 휴강일

[ ] 시험

[ ] 과제 마감

[ ] 학교 행사

2. 메인 캘린더 (내용 표시 영역) 🗓️
사이드바에서 선택한 주차나 월에 해당하는 상세 내용이 여기에 표시됩니다.

뷰 전환 버튼:

메인 캘린더 우측 상단에 [월간] / [주간] 뷰를 전환할 수 있는 버튼을 배치합니다. 사용자가 사이드바에서 '8주차'를 눌렀다면 자동으로 '주간 뷰'로 보여주는 것이 자연스럽습니다.

헤더(제목) 영역:

현재 보고 있는 뷰의 정보를 명확하게 표시합니다.

월간 뷰: < 2025년 9월 >

주간 뷰: < 8주차: 2025년 10월 20일 ~ 10월 26일 >

주요 기능별 UI/UX 아이디어
1. 15주차 주간 뷰
레이아웃: 일반적인 주간 캘린더처럼 월요일부터 일요일까지 7일의 칸으로 구성합니다.

일정 표시:

시간이 정해진 일정(예: 10:00 보충수업)은 해당 시간대에 막대 형태로 표시합니다.

하루 종일 이어지는 일정(예: 개교기념일)은 상단에 별도의 막대로 표시합니다.

핵심: 각 일정 막대에 과목명과 강의실 정보가 바로 보이도록 합니다.

2. 월간 뷰
레이아웃: 격자 형태의 익숙한 월간 달력입니다.

일정 표시: 월간 뷰는 공간이 좁으므로 모든 정보를 표시하기 어렵습니다. 따라서 핵심 정보만 시각적으로 전달해야 합니다.

색상 점(Dot) 또는 캡슐: 일정의 종류(휴강일, 시험 등)에 따라 서로 다른 색상의 작은 점이나 캡슐 형태로 날짜 아래에 표시합니다.

배경색 강조: 중간고사 기간, 축제 기간 등 여러 날에 걸친 중요한 이벤트는 해당 날짜들의 배경색을 은은하게 바꿔서 기간 전체를 강조합니다.

휴강일 처리: 공휴일, 개교기념일 등 쉬는 날은 날짜 숫자를 빨간색으로 표시해 직관적으로 알 수 있게 합니다.

상세 정보: 날짜 칸을 클릭하거나 마우스를 올리면(Hover) 그날의 전체 일정이 작은 팝업(Popover) 창으로 뜨도록 하여 상세 내용을 확인할 수 있게 합니다.

3. 일정의 시각적 구분 🎨
사용자가 일정을 헷갈리지 않도록 색상과 아이콘을 적극적으로 활용하는 것이 중요합니다.

일정 종류	추천 색상	아이콘 (예시)	설명
휴강일/공휴일	빨간색 계열	🏖️	가장 눈에 잘 띄어야 하는 중요한 정보입니다.
시험	파란색/남색 계열	✍️	긴장감과 중요도를 나타내는 색상입니다.
과제 마감	주황색/노란색 계열	🔔	주의를 환기시키는 색상입니다.
학교 행사	초록색 계열	🎉	긍정적이고 활동적인 느낌을 줍니다.
보강	보라색 계열	✨	기존 수업과 구분되는 특별한 일정입니다.
